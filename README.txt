== Description ==

This plugin allows your customers to enter their booking references with password in order to see their Itineraries

== Installation & Setup ==

Typically one of our team members would send you the installation .zip file.

- In the WordPress dashboard under 'Plugins' menu, click the 'Add New' button.
- Click on 'Upload Plugin', choose the file we've sent you and click 'Install Now'
- Our plugin should appear in the 'Installed Plugins' page. Please make sure it's activated.
- Go to 'Sings Itinerary' page and paste in your 'Sings Widget ID' in the existing field. You can find your Widget ID by
going to your branch setup in SinGS.
- Widget is available for you to place anywhere on your website. You can add it in the 'Widgets' page in dashboard.
